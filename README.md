# Animal Corner 🐾

Pet Profiling and Veterinary Appointment Management Mobile Application.

## 📱 Overview

Animal Corner is a mobile application designed to help pet owners manage their pets' profiles and schedule veterinary clinic appointments easily.

The system improves the efficiency of pet profiling and clinic scheduling while providing a convenient platform for pet owners.

This application was developed as part of our undergraduate capstone project.

---

## ✨ Features

- Pet Profile Management
- Upload Pet Images
- Appointment Scheduling
- User Authentication
- Veterinary Clinic Records
- Email Verification
- Firebase Integration
